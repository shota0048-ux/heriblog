## 台本の使い方

### 基本構成（共通）
- **0–3秒：フック**（強い問いかけ・意外な事実）
- **3–15秒：問題提起**
- **15–50秒：本編説明**
- **50–60秒：結論＋CTA**（ブログ誘導）
- **60秒以降：締め**

### 編集のヒント
- ナレーションは**ゆっくり明瞭に**（AI音声でも自分声でもOK）
- テロップは**3〜4行までで簡潔に**
- 効果音（パキッ、ドン、警報音）でメリハリ
- 1秒に**1.5文字程度**のテロップ密度が読みやすい

### 推奨ツール
- **編集**：CapCut（無料、Shorts最適化テンプレ豊富）／ Adobe Premiere
- **AI音声**：VOICEVOX（無料）／ にじボイス／ ElevenLabs（高品質）
- **素材**：Pixabay、Pexels（フリー素材）／ Canvaのストック動画
- **音楽**：YouTube Audio Library（著作権フリー）

## 投稿戦略

### 頻度
**週1〜2本**（最初は無理せず週1）

### 投稿時間
**夜18:00〜21:00**（勤務後に視聴される時間帯）

### ハッシュタグ
共通タグ：`#ヘリコプター #パイロット #航空 #shorts`
個別タグ：内容に応じて `#航空安全 #着氷 #ヴァーティゴ` 等を追加

### サムネイル
- Shortsは最初の1秒でつかむため、**0:00のフレームを意識**
- 大文字テロップ＋強い背景画像

### コメント対応
- **最初の48時間**でコメントに返信すると伸びやすい
- 「ブログ書いてるパイロットです」と一言入れて誘導

## ブログ⇔ YouTube の循環

```
ブログ記事を書く
  ↓
要点を抜き出して台本化（このフォルダ）
  ↓
Shorts投稿（説明欄からブログへ誘導）
  ↓
コメント・反応 → 次の記事ネタへ
```

この循環ができれば、**ブログとYouTubeで相互送客**しながら両方が伸びていきます。
